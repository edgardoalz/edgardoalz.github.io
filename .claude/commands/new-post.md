---
description: Create a new blog post with proper frontmatter
---

Create a new blog post in the `_posts/` directory with the following:

1. Filename format: `YYYY-MM-DD-title.md` (today's date)
2. Required frontmatter:
   ```yaml
   ---
   layout: post
   title: "Post Title"
   date: YYYY-MM-DD HH:MM:SS
   categories: []
   ---
   ```
3. Ask me for the post title and categories
4. After creating the post, run `make rebuild` to update the search index
5. Provide the URL path where the post will be accessible

Use Spanish for the content unless otherwise specified.
